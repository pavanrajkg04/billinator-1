# Public Assets

This directory contains static assets that are served by Next.js.

## Logo

Place your Billinator logo file here as `logo.png` or `logo.svg`.

The logo should be:
- **Format**: PNG (recommended) or SVG
- **Filename**: `logo.png` or `logo.svg`
- **Recommended size**: 
  - For navigation: 32x32px to 64x64px
  - For hero section: 128x128px to 256x256px
  - For footer: 48x48px to 64x64px

The logo is used in:
- Landing page (navigation and hero section)
- Login page (navigation)
- Register page (navigation)
- Dashboard and all app pages (sidebar)
- Footer

## Current Logo

The logo should feature:
- A stylized alligator/crocodile head (dark teal/green)
- A document/bill beneath it
- The alligator "biting" into the document
- Dark teal/deep green color scheme

Make sure the logo has a transparent background for best results on dark themes.
